Import

This section describes how to import statements into pydosh.

From the options menu at the top, click on Import and you'll be prompted with a file dialog. Select the CSV format file or files you wish to import and select the relevant account type from the bottom drop down.

pydosh will then read the file(s) and present what it's found to you.

Records that can be imported will be marked as Ready. Records it doesn't understand and marked as Error. Baks tend to add a lot of crap to their statements, so you'll probably see a few errors which can probably be just ignored.

You will never import a record twice, so don't worry if transactions overlap in statements or if you've imported a statement twice. Transactions that exist  will be flagged as Imported.

Only records that are marked with Ready will be imported, so it's probably safe just to press the Select All button, followed by the Import button.
