This section describes how to login pydosh and how to create a new database.

Logging into pydosh

pydosh needs a Postgres database in order to store your bank transactions.

createdb --host localhost --username bob pydosh
See here for more details.

pydosh will detect an database is empty and initialise it for you. Press the Connect button and it will ask you if you want to initialise it. This assumes that the database has already been created!
