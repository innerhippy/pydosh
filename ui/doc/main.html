Main Window

This is the main window for using pydosh. 

Once you have imported your bank statements, the main window will present all the transactions



The first colum Check indicates if the record has been validated (checked). The second indicates if the record has been tagged - indicated by a icon. The other columns are the fields from the statements. Credit (money in) is shown in green, debits as red.

The bottom line just gives you information about the database connection, total money in and out, and number of records shown / number of records in total.

There are a few buttons that are hopefully intuitive

select most recent month only
toggles the checked flag for selected records
reload all data and reset filters
tag selected records. See tags for more info on tags

Filtering Records

Filtering can be done in a few ways, according to what you choose in the top three boxes.

Date

The date filters can specify a date range or click the checkbox to just show records from the start date field, eg just show Jan 2010. The button will just select the most recent month.

Filter Filter

The second box has three dropdowns, the show dropdown selects to show unchecked records only, checked only or all. The account dropdown will filter records for a particular account. The type dropdown filters on the transaction type from the 4th column.

Search

This is probably the most used filter. Type something into the description box and only records that contain this word or phrase are displayed. The filtering will occur for each character entered (case is insensitive).

The amount field filters on the amount colum - this is probaby the easiest way to fins a transaction, especially when the filter is set to view only checked records.

Basic maths operators can be supplied to the amount field: > < >= <=.
For example, to search for all transactions over ï¿½100 (or $100...), enter ">100"

The Tag field are discussed in the next section.

Tags

Somtimes it is useful to assign arbritrary labels or tags to transactions. For example if you want to tag a bunch of records that are all to do with utililty bills, then you assign a tag called utility by selecting the relevant record (or records) and pressing the  button. This example uses the NPOWER GAS record.

This will present a window that initially will have no tags. Press the button and enter "utility" in the prompt window. This will create a new tag. Now assign this tag by clicking the checkbox,



Press Save, and then you'll see that a tag has appeared next to the NPOWER GAS record.



To use the tags in a filter, just enter the tag you wish to find in the tag box. The box will auto-complete when you write to try and find your tags correctly.
